This is the motherboard of the computer. The motherboard is one of the most essential pieces of the computer. It holds together a variety of crucial components to a computer such as the cpu, memory and connectors for input and output devices.

    Components of a Motherboard

  • A CPU socket
  • A power connector
  • Slots for the systems main memory
  • A chipset
  • Several connectors
  • Slots for hard drives
  • ROM chip
  • A slot for a video graphics card
  • Additional slots for PCI

CPU stands for central processing unit. The CPU is the brains of the computer which is the most important part of the computer system.

    Components of a CPU

  • The arithmetic logic unit(ALU)
  • The control unit(CU)

Ram stands for random access memory. A RAM disk or RAM Drive is what gives the computer amount of memory space that the computer can hold . It also allowes the computer to handle more programs simultaneously.
